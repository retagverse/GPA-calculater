# GPA Calculator (C++)

## Overview
This project is a console-based GPA Calculator built using C++. The application converts letter grades into GPA points and calculates the overall Grade Point Average (GPA).

## Features
- Enter grades for multiple subjects.
- Converts letter grades to GPA points.
- Calculates the overall GPA.
- Displays the GPA with two decimal places.
- Provides a performance evaluation.

## Grading Scale

A+ = 4.0
A  = 3.8
A- = 3.6

B+ = 3.4
B  = 3.2
B- = 3.0

C+ = 2.8
C  = 2.6
C- = 2.4

D+ = 2.2
D  = 2.0

F  = 0.0
